Welcome to the Angular quickstart project of BA.

Run "npm i" to install all the dependencies. Make sure you download the Redux devtools exstension for your browser to see the NGRX store included in this project!

This project includes:

  • Latest Angular version
  • Latest NGRX version
  • NGRX Devtools
  • NGRX Effects
  • Component structure
  • Styling structure
  • Example styling such as mixins and variables

Structure explained

The structure you're seeing may be new but it's my own creation. It's the same for Angular components and styling really quite simple:

  • Pages are made out of layouts
  • Layouts are made out of blocks
  • Blocks are made out of components

So for example:

  • The homepage is made out of layouts: header, footer, main
  • The header is made out of blocks: main navigation, secondary navigation
  • The main navigation is made out of components: logo, search-bar, account-link etc.
